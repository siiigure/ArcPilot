# ArcPilot 社区 + AI 助手 — 逻辑数据模型

本文档将简化领域模型细化为可落地的逻辑模型，包含：Django 字段类型、约束（unique / db_index）、级联删除策略，以及 Session/Message 的 metadata 语义、引用来源（URL vs Object Storage）设计。

---

## 1. 元数据语义约定

| 模型 | 字段名 | 含义 | 使用规则 |
|------|--------|------|----------|
| **AiSession** | `environment_preset` | **环境预设**：会话创建时或用户预设的 ArcGIS 版本、CRS、区域、行业等，整场会话默认生效。 | 发起对话时由前端/插件传入或从用户档案带出。 |
| **AiMessage** | `instant_override` | **即时修正**：用户在某条消息里临时改口（如「我现在换成 ArcGIS Pro 3.2 了」），仅对该条及后续上下文生效。 | 下游合并逻辑：**同名字段以 Message.instant_override 覆盖 Session.environment_preset**，未出现的字段沿用 Session。 |

因此「智能」体现在：同一 Session 内，不同轮次可以有不同的有效环境（例如前几轮是 Pro 3.1，后面变成 3.2），RAGFlow 调用时传入的是「合并后的 metadata」。

## 4. AI 会话与消息（环境预设 + 即时修正）

### 4.1 AiSession

| 逻辑字段 | Django 类型 | 约束 | 说明 |
|----------|-------------|------|------|
| id | AutoField | PK | |
| user_id | ForeignKey(User) | on_delete=**CASCADE**, db_index=True | 用户删则会话删 |
| title | CharField(200) | blank=True | 可选会话标题 |
| created_at | DateTimeField | auto_now_add=True, db_index=True | |
| updated_at | DateTimeField | auto_now=True | |
| **environment_preset** | JSONField | default=dict, blank=True | **环境预设**。建议结构：`{ "arcgis_pro_version": "3.1", "python_env": "conda:arcpy39", "project_crs": "CGCS2000", "region": "CN", "industry": "surveying" }` |

- 级联：AiMessage.session_id → CASCADE。

### 4.2 AiMessage

| 逻辑字段 | Django 类型 | 约束 | 说明 |
|----------|-------------|------|------|
| id | AutoField | PK | |
| session_id | ForeignKey(AiSession) | on_delete=**CASCADE**, db_index=True | |
| role | CharField(20) | choices=(user, assistant, system), db_index=True | |
| content | TextField | | 正文 |
| created_at | DateTimeField | auto_now_add=True, db_index=True | |
| **instant_override** | JSONField | null=True, blank=True | **即时修正**。与 Session.environment_preset 同结构的子集，如 `{ "arcgis_pro_version": "3.2" }`；合并时覆盖 Session 同名字段。 |

- 业务逻辑：调用 RAG 时，对当前 message 及之前轮次做「Session.environment_preset 与每条 Message.instant_override 按时间顺序合并」，得到该轮有效 metadata 再传给 RAGFlow。

### 5.2 AiCitation（参考文献；URL + Object Storage 并存）

行业标准常为 PDF 或内部文档，需支持：**公网 URL** 与 **对象存储 Key（MinIO/S3）**。

| 逻辑字段 | Django 类型 | 约束 | 说明 |
|----------|-------------|------|------|
| id | AutoField | PK | |
| answer_id | ForeignKey(AiAnswer) | on_delete=**CASCADE**, db_index=True | |
| doc_id | CharField(128) | db_index=True | RAGFlow 文档/块 ID |
| title | CharField(500) | blank=True | 展示标题 |
| source_type | CharField(30) | db_index=True | 如 standard, manual, internal_pdf |
| **source_kind** | CharField(20) | choices=('url', 'object_storage'), db_index=True | 来源类型：外链 vs 自建存储 |
| **url** | URLField(1000) | null=True, blank=True | 当 source_kind='url' 时使用（外链） |
| **storage_backend** | CharField(32) | null=True, blank=True, db_index=True | 如 'minio', 's3' |
| **bucket** | CharField(255) | null=True, blank=True | 桶名 |
| **object_key** | CharField(1024) | null=True, blank=True, db_index=True | 对象键，如 `standards/2024/GB50026-2020.pdf` |
| file_name | CharField(255) | null=True, blank=True | 原始文件名，便于展示与下载命名 |
| content_type | CharField(128) | null=True, blank=True | 如 application/pdf |
| standard_code | CharField(100) | null=True, blank=True, db_index=True | 如 GB50026-2020 |
| region | CharField(32) | null=True, blank=True, db_index=True | 适用地区 |
| published_at | DateField | null=True, blank=True | 规范发布/修订日 |
| version | CharField(64) | null=True, blank=True | 版本号 |
| flags | JSONField | default=dict, blank=True | 校验层打的标记，如 suspicious, outdated |

- **使用规则**：
  - 外链（标准官网、外部 PDF 链接）：`source_kind='url'`，填 `url`，其余 storage 字段可空。
  - 内部/上传 PDF（MinIO/S3）：`source_kind='object_storage'`，填 `storage_backend`、`bucket`、`object_key`，可选 `file_name`、`content_type`；前端/后端通过存储后端生成预签名 URL 供下载或预览。

## 7. 级联与索引小结

| 父 | 子 | on_delete | 说明 |
|----|----|-----------|------|
| User | Invite.created_by | CASCADE | 创建者删则邀请作废 |
| User | Invite.used_by | SET_NULL | 保留审计 |
| User | Post.author | SET_NULL | 帖子保留 |
| User | Reply.author | SET_NULL | 回复保留 |
| User | Like | CASCADE | 点赞记录可随用户删 |
| User | AiSession | CASCADE | 会话随用户删 |
| User | AiUsageQuota | CASCADE | 用量随用户删 |
| Post | Reply | CASCADE | 帖子删则回复删 |
| AiSession | AiMessage | CASCADE | 会话删则消息删 |
| AiMessage | AiAnswer | CASCADE | 消息删则答案删 |
| AiAnswer | AiCitation | CASCADE | 答案删则引用删 |

- **db_index** 已在上表对常用查询字段标出（外键、状态、时间、code、slug、source_kind、object_key 等）。
- PostGIS：仅 `Post.project_geom` 使用 GeometryField(srid=4326)；若需空间索引可在该字段上加 `spatial_index=True`（Django/PostGIS 支持）。

## 8. 合并 metadata 的推荐实现（伪代码）

```text
def get_effective_metadata(session: AiSession, up_to_message: AiMessage) -> dict:
    base = dict(session.environment_preset or {})
    for msg in session.messages.filter(created_at__lte=up_to_message.created_at).order_by('created_at'):
        if msg.instant_override:
            base.update(msg.instant_override)
    return base
```

这样「用户中途说换成 Pro 3.2」会在该条及之后所有轮次中生效，满足「即时修正覆盖环境预设」的智能行为。
